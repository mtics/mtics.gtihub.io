总结2019，展望2020
Created 2020-03-02| Updated 2020-05-23|随笔
Post View:|Comments:

本来应该元旦前就写好的总结，一直拖到了今天。没办法，研究生考试结束了又赶上期末结课和考试以及毕设相关事宜，元旦后放了三天假又忙了三天，就一直拖到了今天才开始写。

总结篇

  1. 2019年上半年随队参加了Robocup中国赛标准平台组，拿下了大学三年来自己的最高奖项——中国赛季军，随后不久又拿下了亚洲赛亚军。
  2. 2019年下半年主要就在准备考研。因为时间短，数学没能好好复习，但是我可以问心无愧地说一句“尽力了”。考完了之后我屏蔽了所有考研群和公众号，不对答案不参与和讨论。但行好事，莫问前程。

规划篇

做规划得做两手准备，考研上岸了一个规划，没上岸再做一个规划。

未来十年长期计划（2020-2029）

  1. 读研
  2. 读博（可选）
  3. 找一个合适的工作
  4. 结婚
  5. 有自己的房子

2020年计划

这部分就得分为两部分了，我太难了……

未成功上岸版

  1. 锻炼身体
  2. 好好学习，准备考研

成功上岸版

  1. 锻炼身体，减肥（至少到160斤）
  2. 试着翻译一些计算机专业英文材料
  3. 做好专业要做的事，学好该学的习
  4. 掌握一门能挣钱的副业
  5. 学习Java Web/小程序
  6. 日语入门（可选）
